Шапка сайта вверху справа содержит меню пользователя (ссылки на регистрацию, авторизацию и действия с контентом, сделанные с IP-адреса пользователя).
Под ним - меню в виде вкладок (слева вкладки с заглавной страницей и с обсуждением, справа - вкладки для разработчиков статей (читать, просмотр кода, история изменений), а так же окно для поиска.

Слева сайта содержится вытянутый в высоту прямоугольный блок с основным меню взаимодействия с сайтом (для просмотра и правки статей, печати, ссылки на аналогичные проекты и перевод на другие языки). Вверху блока находится эмблема сайта, по клику ведущая на заглавную страницу.

Основной контент состоит из 3 основых частей:

  1. Блок с приветствием (содержит так же небольшое меню справа с ссылками на порталы, справку и страницу для создания статьи).

  2. Содержит несколько прямоугольных блоков:
     - блок "Избранная статья" (фрагмент одной из избранных статей, одно фото);
     - блок "Из новых материалов" (маркированный список из нескольких пунктов на тему "Знаете ли вы?", 2-3 фото);
     - блок "Текущие события" (ссылки на актуальные темы, маркированный список актуальных новостей, одно фото); 
     - блок "Хорошая статья" (небольшой начальный фрагмент одной из хороших статей);
     - блок "В этот день" (маркированный список событий, произошедших в этот день в разные годы, одно фото);
     - Блок, в котором показаны последний и предыдущий избранные списки;
     - Блок "Совместная работа недели" (приглашение желающих к совместной работе над статьями Википедии и участию в конкурсах)
     - Блок "Изображение дня" (случайное фото из одной из статей)
     В каждом блоке (кроме двух последний) внизу слева есть кнопка действия в соответсвии с содержанием блока (читать, обсудить, другие текущие события, другие избранные списки).

  3. Блок, содержащий ссылки на другие проекты фонда Викимедиа.
 
Подвал сайта содержит:
   - ссылки на лицензию, условия использования, владельца товарного знака;
   - ссылки на политику конфиденциальности, контакты для связи и прочую дополнительную юридическую информацию о сайте;
   - иконки проектов, помогающих сайту Википедия.
